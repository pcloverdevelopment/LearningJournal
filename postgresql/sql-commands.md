# SQL Commands

## Categories of SQL Commands

- Data Definition Language (DDL)
  - CREATE
  - ALTER
  - DROP
  - RENAME
  - TRUNCATE
  - COMMENT
- Data Querying Language (DQL)
  - SELECT
- Data Manipulation/Modification Language (DML)
  - INSERT
  - UPDATE
  - DELETE
  - MERGE
  - CALL
  - EXPLAIN PLAN
  - LOCK TABLE
- Data Control Language (DCL)
  - GRANT
  - REVOKE
